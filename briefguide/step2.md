# Step2 创建子网

1）登录控制台，选择【产品与服务】中“私有网络VPC”，进入私有网络页面。

2）可在子网标签中点击“创建子网”按钮创建子网实例，创建时需要填写以下信息：

``` 
    * 所属VPC：选择将子网创建在哪一个VPC中。
    * 所属网段：选择VPC中已添加的网段。
    * 子网名称：输入长度为6~63位的名称。
    * 子网网段：根据VPC的网段，选择子网的网段。同一个VPC中的子网网段不能重叠
```

3）创建成功后，可在子网页面查看所创建的子网实例，并对其进行管理，例如更改备注、更改业务组、管理资源等。
